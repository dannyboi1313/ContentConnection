Welcome to Analogy

How to Play:

The premise of "Analogy" is to guess the connection between two given media types. For example, if two pictures are given, the object of the game is to guess the analogous relationship between the two. The connection is typically somewhat abstract.
The user has 5 chances to guess the answer. After two wrong guesses, the user is given a hint. After 4 wrong guesses, the user is given one final hint. 5 wrong guesses is a loss. Similar to "Wordle", the user can only play this game once per day.
